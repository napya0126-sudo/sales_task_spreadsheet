# 営業からのタスク管理スプレッドシート

Google スプレッドシートと Google ドキュメントを連携させた、営業チーム向けのタスク管理システムです。

## 主な機能

1.  **タスクの自動補完**:
    - 「全て」シートの B列（タスク名）を入力すると、依頼者、ステータス（未着手）、期限（7日後）が自動的に入力されます。
2.  **担当者別シートの同期**:
    - 依頼先（E列）を選択すると、担当者ごとの個別シートが自動作成/更新され、ステータスやメモが双方向に同期されます。
3.  **Google ドキュメント連携（新機能）**:
    - スプレッドシート上のタスク名を、指定した Google ドキュメントの末尾に「見出し」として自動追加します。
    - 作成された見出しへの直接リンク（アンカー付きURL）を H列（Doc URL）に自動的に書き戻します。
4.  **アーカイブ機能**:
    - 完了したタスクを一括でアーカイブシートに移動し、メインシートをクリーンに保ちます。

## セットアップ

### 環境構築
このプロジェクトは [clasp](https://github.com/google/clasp) を使用して管理されています。

```bash
# インストール
npm install -g @google/clasp

# ログイン
clasp login

# プッシュ
clasp push
```

### Google Docs API の有効化
Google ドキュメントの見出し ID を取得するために、Google Apps Script の「サービス」から **Google Docs API** を追加する必要があります。

1.  GAS エディタを開く。
2.  左側の「サービス +」をクリック。
3.  「Google Docs API」を選択して追加。

## 使い方（Docs 連携）

1.  スプレッドシートの「全て」シートに新しいタスクを入力します。
2.  上部メニューの **「タスク管理」＞「🔗 Docに見出しを作成」** をクリックします。
3.  自動的に **B列の最終行（最新タスク）** が判別され、以下の処理が行われます：
    - 連携先ドキュメントの末尾に「見出し 3」が追加される。
    - その見出しへのリンクが H列に書き込まれる。

## 設定

`コード.js` 内の以下の定数を環境に合わせて変更してください。

- `TARGET_DOC_ID`: 連携先の Google ドキュメント ID
- `PERSONS`: 担当者のリスト
- `EMAIL_TO_NAME`: メールのプレフィックスと表示名のマッピング
